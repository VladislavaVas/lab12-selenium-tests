# lab12-selenium-tests
Автоматизированные тесты для лабораторной работы №12.
Описание тест-кейсов:
Тест 1. Успешный вход
Проверка авторизации с корректными учётными данными (`standard_user` / `secret_sauce`). Ожидается, что после входа открывается страница каталога товаров и отображается список товаров.

Тест 2. Попытка входа заблокированного пользователя 
Проверка, что пользователь `locked_out_user` не может авторизоваться. Ожидается появление сообщения об ошибке «Epic sadface: Sorry, this user has been locked out.», и пользователь остаётся на странице входа.

Тест 3. Добавление товара в корзину
Проверка добавления товара «Sauce Labs Backpack» в корзину после успешного входа. Ожидается, что счётчик товаров на иконке корзины становится равным 1, и при переходе в корзину там отображается выбранный товар.
